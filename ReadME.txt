Designed custom based Web Crawler by extending open source Crawler4J Web Crawler to search the Website URL in automated regular manner in fixed intervals and write the contents to DAO - Database Object - SQL Server - JDBC-ODBC Driver 4 connection is used.

The crawled web page is www.ratemyprofessor.com which fetches the professor details - University name , Professor name , Ratings , Comments , Feedback and writes to the DAO object . The Schedular is used to implement for recrawling for updates and subsequently updating the Database with related content.

The crawler built follows robots.txt norms
